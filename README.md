# digital_order

このアプリは小規模飲食店向けの伝票アプリで、紙媒体を使用している店舗からの移行を想定しています。
マルチプラットフォームアプリなので、環境を選ばずにご利用いただけます。
また、データベースや決済サービスと連係することにより、
さらに会計をスマートにし、その後の売上確認などにも貢献します。
